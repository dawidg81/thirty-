# ccraft
ccraft is a custom Minecraft Classic server software that is going to support Minecraft Classic Server Protocol.

Inspirations taken from:
Thirty by Sean Baggaley: http://dev.firestick.games/sean/thirty

ccraft is going to be written in C++ and will support Minecraft Classic Server Protocol (its documentation can be found here: [1], [2], [3])
